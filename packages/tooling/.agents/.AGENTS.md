# AGENTS.md (Tooling)

Inherits global policy from [/.agents/AGENTS.md](../../../.agents/AGENTS.md).

## Scope

Applies to everything under `packages/tooling`.

## Package Role

`tooling` provides shared lint/format configuration consumed across the monorepo (specifically, stylelint configurations since it is out of scope for oxlint).

## Local Rules

1. Keep config changes minimal and predictable across all packages.
2. Prefer backward-compatible config updates.
3. Verify that rule changes are justified and scoped to the request.
4. Avoid package-specific assumptions in shared config exports.

## Commands

From repo root:

- `vp run tooling#lint:style`
- `vp run tooling#lint:style-fix`

Monorepo verification after tooling changes:

- `vp run check-all`

## Validation

For tooling changes, run at least:

1. `vp run check-all`

## Avoid

- Broad rule churn unrelated to the task
- Silent formatting/style shifts across the monorepo without request
